Comportement

Au clic : le prompt est copié dans le presse-papier en silence, puis la plateforme cible s'ouvre dans un nouvel onglet avec le prompt pré-rempli en paramètre URL. Un toast discret confirme le succès. Si la plateforme ne lit pas le paramètre URL (cas rare), l'utilisateur n'a qu'à coller (Cmd+V ou Ctrl+V) pour utiliser le prompt déjà copié.
